为了可视化 Sugarscape 实验中的“涌现”现象，我计划编写一个专门的仿真与绘图脚本。该脚本将运行模拟并生成四张关键图表，以展示微观规则如何导致宏观的社会结构。

### 1. 依赖安装
- 需要安装 `matplotlib` 用于绘图。

### 2. 编写可视化脚本 `scripts/visualize_sugarscape.py`
该脚本将执行以下步骤：
1.  **运行仿真**：运行 Sugarscape 模拟 500 个时间步（tick）。
2.  **数据采集**：
    -   记录每一帧的 Gini 系数。
    -   记录初始状态（t=0）和最终状态（t=500）的所有 Agent 位置和财富数据。
3.  **生成图表**：
    -   **图 1：财富分布演变 (Wealth Distribution)**
        -   **类型**：直方图 (Histogram)
        -   **目的**：对比初始时刻（均匀分布）与结束时刻（高度偏斜/长尾分布）的财富分布，展示贫富分化的涌现。
    -   **图 2：不平等曲线 (Lorenz Curve)**
        -   **类型**：洛伦兹曲线
        -   **目的**：展示财富集中的程度（例如“20%的人拥有80%的财富”）。
    -   **图 3：不平等随时间变化 (Inequality over Time)**
        -   **类型**：折线图
        -   **目的**：展示 Gini 系数随时间的变化趋势，观察系统是否达到稳态。
    -   **图 4：空间聚集模式 (Spatial Clustering)**
        -   **类型**：散点图 + 等高线图
        -   **目的**：在糖山地形图上叠加 Agent 的位置。对比 t=0 和 t=500，观察 Agent 是否向富糖区聚集（资源驱动的迁徙）。

### 3. 目录结构
- 创建 `results/sugarscape/` 目录用于存放生成的图片。

### 4. 执行
- 运行脚本并向您展示生成的图片路径。
